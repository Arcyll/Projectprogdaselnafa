PROJEK PROGDAS SAHABAT BUKU
by
- Rira
- Elissa
- Nabila

Aplikasi “Sahabat Buku” adalah sarana bagi para 
donatur, pengelola, dan penerima agar dapat melakukan donasi 
dan distribusi buku dengan mudah dan tersentralisasi, di mana semua 
informasi yang dibutuhkan dapat diakses oleh masing-masing stakeholder.

Cara kerja pengguna:
- Pilih mode user
- Isi data diri
- Donasikan buku
